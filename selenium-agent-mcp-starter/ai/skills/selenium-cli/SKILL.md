# Selenium CLI Skill

Use this skill when you need browser-level guidance for the Selenium TypeScript starter.

## Purpose

This skill helps an agent:

- translate interactive browser exploration into Selenium-friendly selectors
- recommend waits and interaction patterns
- turn a discovered flow into task-map steps or page object methods

## Use This Skill For

- login flows
- navigation flows
- resilient selector design
- screenshot/debug guidance
