# Prompt Library

Use these prompts to guide AI-assisted work in the Selenium TypeScript starter.

## Test Design

```text
Design Selenium test cases for the login flow using the existing page objects first. Keep the suite small, stable, and readable.
```

## Selector Review

```text
Review the selectors in the page objects and suggest more stable alternatives where they are overly brittle.
```

## Agent Map Generation

```text
Convert this manual test flow into the JSON structure used by agent-map.json, including assertions and screenshots where useful.
```

## Flake Triage

```text
Analyze this failing Selenium test and tell me whether the root cause is timing, selector stability, environment drift, or application behavior.
```
